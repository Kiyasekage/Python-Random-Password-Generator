# Python-Random-Password-Generator
*Description
This Python program generates a random password based on a user-defined length.
The program asks the user for a username and the desired password length. It then creates a password using a combination of lowercase letters, uppercase letters, and digits.
To ensure basic password strength, the generated password always contains at least:
one lowercase letter
one uppercase letter
one digit
After generating the password, the program displays it to the user.
This project demonstrates the use of Python libraries, random selection, and string manipulation to create a simple password generation tool.

*Features
Generates a random password
User can choose the password length
Ensures the password contains lowercase letters, uppercase letters, and digits
Uses Python's random and string modules
